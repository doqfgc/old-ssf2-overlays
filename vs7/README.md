# Season 7 Overlay
This overlay version is used during ACX Season 7. It was first used at *Flash Summit: Summit of Kings* on 24 August 2025. It is built on CoreOverlay v1.1.1 and is usable nearly unmodified.

This is the current overlay. There is no scheduled release for this overlay yet.
